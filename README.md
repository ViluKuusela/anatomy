# Anatomy Quiz App

Android app for learning human anatomy (currently bones).

## Features
- Quiz-based learning
- Three quiz modes: 1) tap the correct bone 2) choose the name of the bone 3) write the name of the bone
- Focus on skeletal system
- Simple and fast UI
- App language support for English and Finnish
- Anatomy language support for English, Latin and Finnish

## Tech
- Android Studio
- Kotlin / Java
